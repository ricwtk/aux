Provide your student ID mdi-format-list-bulleted Any change to the student ID will reset the data on the page. Double and triple check to make sure it is correct and do not require changes later. Click mdi-pencil to modify student ID mdi-format-list-bulleted Be reminded: any change to the student ID will reset the data on the page. Double and triple check to make sure it is correct and do not require changes later. Click mdi-plus to add the root node mdi-format-list-bulleted Root node mdi-format-list-bulleted Any new node added will have the state of "-1,-1". Edit node mdi-format-list-bulleted Move your cursor to the node your want to edit, and click on mdi-pencil to edit information of the node. Add child mdi-format-list-bulleted Move your cursor to just below the parent node, and click on mdi-plus to add a child to the node. New child mdi-format-list-bulleted A new child will have the state of "-1,-1" and action of "x". Multiple children mdi-format-list-bulleted Additional children can be added in the same way. Node information dialog mdi-format-list-bulleted Clicking on mdi-pencil on a node brings up the node information dialog. Edit state mdi-format-list-bulleted This box contains the state of the node. Any string is allowed. Edit expansion sequence mdi-format-list-bulleted Expansion sequence is the order of the nodes being expanded. -1 for unexpanded node. 1 for the first expanded node, which should be the root node. 2 for the second expanded node, and so forth. Display of the expansion sequence of a node mdi-format-list-bulleted Removed node mdi-format-list-bulleted A removed node is normally a node that leads to redundant path. Display of a removed node mdi-format-list-bulleted Assign action to child mdi-format-list-bulleted This section allows you to change the action label for the children of the node. Click mdi-arrow-up-bold or mdi-arrow-down-bold to change the order of the children. Delete node mdi-format-list-bulleted Click deleteto delete the node. All successors (children, grandchildren, ...) will also be deleted. When you are done mdi-format-list-bulleted Click Download to download the json file. Submit the downloaded file through the submission link. What if my computer crashed? mdi-format-list-bulleted If there is a risk of your computer being crashed, remember to download the file from time to time. Click Upload to view the upload file dialog. Click the text field to select the json file that you have previously downloaded. Click Upload to upload the file. This is not for your submission. Submit through the submission link provided for the assessment.
