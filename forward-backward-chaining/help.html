Provide your student ID mdi-format-list-bulleted Any changes to the student ID will reset the data on the page. Double and triple check to make sure it is correct and do not require changes later. Select the process you will be conducting mdi-format-list-bulleted Select forward or backward chaining process. This can be modified easily later. Click OK to start providing your solution. Set the parameters of the problem mdi-format-list-bulleted Select the number of possible facts in your problem. The facts will be rendered as alphabets. For example, if you specify the number of possible facts to be 5, the facts you get are A, B, C, D, E. Select the number of rules in your problem. The rules will be rendered starting from rule 1, rule 2, and so forth. Click mdi-pencil to modify student ID and/or process. mdi-format-list-bulleted Initial condition of your production system mdi-format-list-bulleted The initial facts in the working memory will be defined for you and you do not need to modify this part. Next step mdi-format-list-bulleted Click mdi-plus to add new step. Working memory mdi-format-list-bulleted This box should contain all facts available in the current working memory. Click in the selection box to add or remove facts from working memory. Add fired rule mdi-format-list-bulleted Click mdi-plus to add new fired rule. Add fired rule mdi-format-list-bulleted Select the rule that has been fired. Make sure you add the new fact(s) into the working memory. The rules listed here are the rules being fired from the beginning of the forward/backward chaining process in sequence. Fired rule mdi-format-list-bulleted For example, if rule 3 is fired in step 1, and rule 1 is fired in step 2, you should have this in your step 2. Insert or remove fired rule mdi-format-list-bulleted Click mdi-plus to insert a rule above the rule. Click mdi-minus to remove the rule. Add goal/subgoals (Backward chaining) mdi-format-list-bulleted For backward chaining process, there is the additional section to add goal/subgoals. Click mdi-plus to add a goal or subgoal to the end of list. Add goal/subgoals (Backward chaining) mdi-format-list-bulleted Click the left selection box to choose the goal or subgoal. Click the right selection box to choose the rule that will achieve the goal or subgoal. Leave the right selection box blank if the rule to achieve the goal or subgoal has not been identified. Click mdi-minus to remove the goal or subgoal. Remarks mdi-format-list-bulleted A remark is optional. You may use this to explain your step if you wish to. List of steps mdi-format-list-bulleted Click mdi-plus to insert a step. Click mdi-minus to remove the step. Click mdi-pencil to edit the step. When you are done mdi-format-list-bulleted Click Download to download the json file. Submit the downloaded file through the submission link. What if my computer crashed? mdi-format-list-bulleted If there is a risk of your computer being crashed, remember to download the file from time to time. Click Upload to view the upload file dialog. Click the text field to select the json file that you have previously downloaded. Click Upload to upload the file. This is not for your submission. Submit through the submission link provided for the assessment.
