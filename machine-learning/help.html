Provide your student ID mdi-format-list-bulleted Any changes to the student ID will reset the data on the page. Double and triple check to make sure it is correct and do not require changes later. Click OK to start providing your solution. Click mdi-pencil to modify student ID mdi-format-list-bulleted Create a new text cell mdi-format-list-bulleted Click Add text to create a new text cell. Each block of information you are providing is called a cell. Text cell mdi-format-list-bulleted A text cell is used to provide textual information. Create a new empty table mdi-format-list-bulleted Click Add empty table to create a new empty table cell. Table cell mdi-format-list-bulleted A table cell is used to provide tabular information. The first row (shaded) is the header of the table, i.e. the name of the columns. Click mdi-table-column-plus-after to create an empty column to the right. Click mdi-table-row-plus-after to create an empty row below a row. Click mdi-table-column-remove to remove a column. Click mdi-table-row-remove to remove a row. Insert the raw data mdi-format-list-bulleted Click Insert raw data to insert the raw data. Raw data is the data provided by the question. Normally provided as a table. Raw data mdi-format-list-bulleted Generic cell controls mdi-format-list-bulleted Controls of a cell are at the top of the cell. Click mdi-content-copy to copy a cell. Click mdi-chevron-up to move a cell up. Click mdi-chevron-down to move a cell down. Click mdi-minus to remove a cell. Paste the copied cell mdi-format-list-bulleted When a cell is copied, Paste copied cell will be available. Click Paste copied cell to insert the copied data. When you are done mdi-format-list-bulleted Click Download to download the json file. Submit the downloaded file through the submission link. What if my computer crashed? mdi-format-list-bulleted If there is a risk of your computer being crashed, remember to download the file from time to time. Click Upload to view the upload file dialog. Click the text field to select the json file that you have previously downloaded. Click Upload to upload the file. This is not for your submission. Submit through the submission link provided for the assessment.
