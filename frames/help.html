Provide your student ID mdi-format-list-bulleted Any changes to the student ID will reset the data on the page. Double and triple check to make sure it is correct and do not require changes later. Click OK to start providing your solution. Click mdi-pencil to modify student ID mdi-format-list-bulleted Create a new frame mdi-format-list-bulleted Click mdi-plus to create a new frame. The frame edit dialog will pop up when you create a new frame. Frame name mdi-format-list-bulleted Edit the content of this box to change the name of this frame. Edit slot and slot value mdi-format-list-bulleted Change the name of the slot and the value of the slot by changing the content of the box. Add slot mdi-format-list-bulleted Click mdi-plus to add new slot. Move slot mdi-format-list-bulleted Click mdi-arrow-up-bold to move the slot up. Click mdi-arrow-down-bold to move the slot down. The sequence of the slot is not important. This function is for your own organisation purpose. Delete slot mdi-format-list-bulleted Click mdi-minus to delete the slot. Edit frame details mdi-format-list-bulleted Click mdi-pencil next to the frame name to edit the frame details. Delete frame mdi-format-list-bulleted Click mdi-minus to delete the frame. When you are done mdi-format-list-bulleted Click Download to download the json file. Submit the downloaded file through the submission link. What if my computer crashed? mdi-format-list-bulleted If there is a risk of your computer being crashed, remember to download the file from time to time. Click Upload to view the upload file dialog. Click the text field to select the json file that you have previously downloaded. Click Upload to upload the file. This is not for your submission. Submit through the submission link provided for the assessment.
